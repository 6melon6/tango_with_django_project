## 测试最佳实践

### 1. 测试命名规范

```python
def test_<功能>_<场景>():
    """测试描述"""
    pass
```

**示例**:
- `test_register_success()` - 测试成功注册
- `test_login_invalid_credentials()` - 测试无效凭证登录

### 2. setUp 方法

每个测试类都有 `setUp()` 方法，在每个测试方法前执行：

```python
def setUp(self):
    self.client = Client()
    self.user = User.objects.create_user(username='testuser', password='test123')
    self.category = FoodCategory.objects.create(category_name='测试分类')
```

### 3. 断言使用

```python
# 状态码断言
self.assertEqual(response.status_code, 200)
self.assertEqual(response.status_code, 302)  # 重定向

# 重定向断言
self.assertRedirects(response, '/dashboard/')

# 模板断言
self.assertTemplateUsed(response, 'dashboard.html')

# 数据库断言
self.assertTrue(User.objects.filter(username='test').exists())
self.assertEqual(FoodRecord.objects.count(), 1)

# 内容断言
self.assertContains(response, '今日总卡路里')
self.assertIn('form', response.context)
```
